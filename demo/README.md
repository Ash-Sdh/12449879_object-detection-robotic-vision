\# Demo: YOLOv5 Inference on an Image (CPU)



This demo runs YOLO object detection on a single image and saves an output image with bounding boxes.



\## Requirements

\- Python installed

\- Dependencies installed (`pip install -r requirements.txt`)

\- Ultralytics YOLO CLI working (`yolo` command)



\## Run

From the repo root:



```powershell

python demo\\run\_demo.py --source demo\\assets\\test.jpg
